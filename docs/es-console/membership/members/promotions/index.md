# Promotions

The **Promotions** tab allows a marketer or agent to view the promotions and, if enabled, the badges and banners that are relevant to the current member.

:::info
The **Promotions** tab of the member management page allows you to view **Targeted Offers**. If badges and banners are enabled in the Console, any **Badges** or **Banners** associated with the current member may also be viewed.
:::
