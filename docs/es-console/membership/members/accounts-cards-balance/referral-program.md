# Referral Program

If a referral requires the referee to carry out an activity (such enrollment or enrollment with a minimum spend) to receive rewards, the referral is credited to the referral program active when the referral activity is completed. For example, if a referee receives a referral in Q1, but does not complete the required activity (such as spend $100) until Q2, then that referral is credited to the referral program active in Q2 (assuming that it is different than the Q1 program).

The **Referral Program** section on the member page shows information related to the current member's referral activity, meaning their activities to send referrals to potential new members in the loyalty program. Depending on configuration, the referral program may reward the referrer only (the one providing the code) or the referrer and the referee (the latter either for enrollment only or enrollment and completion of a minimum purchase).

:::info
This section will not appear if configuration is not set up to use Referral Program features or if the user does not have the correct permissions. 
:::

When this section is expanded, the data shown includes:

- **Member's Referral Code** - This is the unique code that the current member can use (or an agent in the Console can use on their behalf) to invite other customers to become members in the loyalty program. Usually, if a new member joins the loyalty program using this code, then completes one or more activities such as registering and completing a minimum spend threshold, then the referrer (or the referrer and referee) will receive rewards. There is a Copy button &#128203; next to the Member's Referral Code that allows an agent in the Console to copy the code.
- **Usage Limited Used** - The referrer will have a limit on how many times they can use the referral code (how many other customers they can invite to become members in the loyalty program). This data is the number of referrals the member has used versus the limit on the total number of referrals they can make (for instance, "1 of 10" where "1" is the current count or number of referrals they have made and "10" is the limit).
