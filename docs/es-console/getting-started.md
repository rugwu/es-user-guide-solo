title: Getting Started with ES Console

---

After you log in to the Console, the first page you will see is the landing page. What is shown on this page depends on how the Console is configured for your company and user profile.

If your company or your user profile do not have access to Insights Pulse, the welcome text ("Welcome to the Exchange Solutions Console") and the image are shown.

![An overview of the ES Console landing page](../../images/Landing%20Page%20without%20Tableau%20Pulse.png)

This view has the Exchange Solutions logo at the top right next to the menu bar (see details below). The copyright notice and environment version information along with links to various policy pages are provided in the footer of this page.

If no Insights Pulse metrics are being followed yet, then a **+Insights** button and some explanatory text is shown. The +Insights button allows the user to access a page from which they can configure and follow their metrics visualizations (the "tiles").

![An overview of the landing page with the Insights button](../../images/Landing%20Page%20with%20+Insights%20Button.png)

Insights Pulse metrics tiles are shown on the landing page only to users who have this feature configured for their company and who also have the correct permissions applied to their user profile. The Insights Pulse elements include any metrics "tiles" or visualizations that the user is following.

![An overview of the landing page with metrics tiles](../../images/Landing%20Page%20with%20Metrics%20Tiles.png)

The top-level menus on the menu bar in the Console are shown depending on the product(s) and configurations with which Console is used and on the permissions associated with the current user. For ES Loyalty™, the full set of possible menus and selections includes:

- **Membership**: Use this menu to find **Members** already in the system and to access **Audiences** consisting of specific groups of members. This menu also provides access to **Scoring and Intelligence** functions to use Gen AI assistance in generating member scores that can be used in **Audiences** or with **Offers**.

- **Promotions**: This menu is used to access and create **Offers** (the value offered to members as part of the loyalty program), **Banners** (the collaterals used to present the offers on your website), and **Badges** (to indicate an achievement for the member). For some Exchange Solutions clients who are using **ES Loyalty Boost™**, the Promotions menu includes additional options under the **Campaign Automation** to allow management of the **Campaign**, **Offer Pool**, or **Seed List**. Clients may also access **Budgets** for their campaigns if they are using Boost. In addition, if configured, users will be able to set up **Games** for their members to play for rewards.

- **Insights**: A **Reports Overview** selection allows the user to access reports in the system. **Report Management** and **Report Identifier** options allow the user to manage their reports and associated identifiers. If enabled, the **Insight Pulse** menu selection provides access to a tool for creating metrics to be shown to the user on the home or landing page of the Console. This provides the Console user with the most critical metrics they have created right after they log in.

- **Program**: This menu allows management of additional features of the loyalty program, but features differ depending on client configuration or setup. Some common additional features are: **Program Tiers** (to manage program member tiers), **Expiry** (to manage member expiry), and **Householding** (to allow members to aggregate their rewards for redemptions).

- **Data**: This menu allows management of data-related aspects of the loyalty program, but features differ depending on client configuration or setup. Some common additional features are: **Activity Identifiers** (activities by which members trigger offers), **Ad Hoc Reward** and **Reward Identifiers** (for loyalty program reports), **Ad Hoc Redemption** and **Redemption Identifiers** (to manage redemption data), **Redemption Catalogue** (to manage catalogue items that can be redeemed at POS and in other channels, sometimes by business unit), and **Extended Member Data** (to manage EMD associated with members).

- **Users**: This menu is used to manage **Users**, **Groups**, and **Roles**. This menu is only available to users with the associated permissions.

The control to **Log out** is also on the menu bar. Click the user profile name in the upper right-hand corner and select **Log out** at any time to log out of the Console.
